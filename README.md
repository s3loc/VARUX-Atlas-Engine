# VARUX-Atlas-Engine
VARUX Atlas Engine, endüstriyel ağları, web uygulamalarını ve güvenlik zafiyetlerini tek merkezden analiz eden birleşik keşif ve istihbarat motorudur. Tüm modülleri orkestre ederek tam görünürlük, hızlı tespit ve kurumsal seviye güvenlik sağlar
